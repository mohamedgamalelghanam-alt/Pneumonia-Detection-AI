# 🩺 نظام التشخيص الذكي للأشعة (Pneumonia Detection AI)

نظام متكامل يعتمد على الذكاء الاصطناعي لمساعدة الأطباء في تشخيص الالتهاب الرئوي بدقة من خلال صور الأشعة (X-ray)، مع تحديد مكان الإصابة بصرياً وتقديم رسائل دعم للمرضى.

## 🔗 روابط المشروع (Project Links)
🚀 **تجربة حية للنظام (Live Demo):**[ [اضغط هنا لتجربة الموقع](https://dd7c5a7f75b55093fa.gradio.live/) ](https://huggingface.co/spaces/mohamed-gamal-ai-eng/Pneumonia-Detection-AI) 

## 🌟 المميزات الرئيسية
- **كاشف الالتهاب الرئوي الذكي 🩺:** نظام يعتمد على الذكاء الاصطناعي لمساعدة الأطباء في تحديد مكان الالتهاب بدقة، مع اليقين بأن الله هو الشافي.
- **التحليل البصري (Grad-CAM):** يقوم النظام بتحديد مكان الالتهاب باللون الأحمر على الصورة لتسهيل عملية الفحص الطبي.
- **تقرير طبي مفصل:** يصدر النظام تقريراً نصياً يشرح الحالة ونسبة التأكد.
- **واجهة مستخدم احترافية:** واجهة مريحة ومنظمة تم بناؤها باستخدام Gradio لضمان تجربة مستخدم سلسة.
- **لمسة إنسانية:** يتضمن النظام رسائل طمأنينة وتوكل على الله لدعم المريض نفسياً خلال رحلة التشخيص.

## 🛠️ الأدوات والتقنيات المستخدمة
- **لغة البرمجة:** Python
- **إطارات عمل الذكاء الاصطناعي:** TensorFlow / Keras
- **واجهة المستخدم:** Gradio
- **مكتبات المعالجة والتحليل:** OpenCV, Matplotlib, Pillow, Numpy

## 🚀 كيفية العمل
1. يتم رفع صورة أشعة الصدر (X-ray) عبر الواجهة.
2. يقوم النموذج بتحليل الصورة وتحديد احتمالية الإصابة بالالتهاب الرئوي.
3. يتم توليد خريطة حرارية (Heatmap) لتوضيح المناطق التي استند إليها النموذج في قراره.
4. يظهر تقرير مفصل يحتوي على النتيجة، نسبة التأكد، ورسالة طمأنينة للمريض.
